# Obesity_Levels_
# 🧠 Obesity Prediction App 🚀

## 📌 Project Overview

This project predicts obesity levels based on lifestyle and health-related factors such as diet, physical activity, and daily habits. The model is built using machine learning and deployed as an interactive web application using Streamlit.

---

## 🚀 Features

* Data cleaning and preprocessing
* Feature engineering (BMI calculation from height & weight)
* Handling multicollinearity
* Multiple model training and comparison
* Hyperparameter tuning
* Final model selection (XGBoost)
* Streamlit web app for real-time prediction

---

## 📊 Models Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest
* Gradient Boosting
* **XGBoost (Final Model)**

---

## 🏆 Final Model

XGBoost was selected as the final model due to:

* High accuracy (~99%)
* Minimal overfitting
* Strong generalization

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Streamlit

---

## 📈 Key Insights

* BMI is the most important feature influencing obesity
* Lifestyle habits such as eating patterns and physical activity play a major role
* Family history also impacts obesity levels
